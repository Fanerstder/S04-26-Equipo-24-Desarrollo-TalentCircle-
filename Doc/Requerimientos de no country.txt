Descripción
TalentCircle publica contenido técnico en múltiples canales: un boletín semanal,
LinkedIn, Twitter y el feed interno de cada comunidad. Actualmente,
un editor compila manualmente el contenido revisando las discusiones,
recursos compartidos y sesiones más activas de la semana.
Este proceso lleva medio día y siempre se pierde contenido valioso.
Expectativas Un pipeline que recopila datos de actividad semanal
de la comunidad (publicaciones con más reacciones, preguntas más respondidas,
recursos más compartidos).
Procesamiento con un LLM para identificar los temas más relevantes y generar resúmenes.
Generación automática de borradores diferenciados por canal:
formato largo para el boletín, formato corto para LinkedIn y formato muy corto para Twitter.
Panel de revisión donde el editor ve, edita y aprueba los borradores para su publicación.
Integración con al menos una API de publicación (API de LinkedIn o similar) o exportación estructurada.
Usuarios Editor de contenido de TalentCircle que revisa y aprueba los borradores generados.
Miembros de la comunidad cuyas contribuciones se amplifican.

Flujo de trabajo El sistema se ejecuta automáticamente todos los viernes al final del día → recopila la actividad de la semana → el LLM analiza y selecciona las contribuciones más relevantes → genera borradores por canal → el editor abre el panel el lunes → revisa cada borrador → edita según sea necesario → aprueba → el sistema publica o exporta según el canal configurado.
